# salesforce-web-to-any-object
Web to any object adds functionality to save a web form data to any Salesforce object with Force.com Sites, similar to Web-to-Lead or Web-To-Case. 

This package requires Force.com sites and administrtaive permissions set to enable any web hosted page to post a form data to service on Force.com site.

Example form is provided and can be hosted on Heroku in directory `heroku-web` thsi is simple PHP app to deploy for testing.
